# 🚀 VOID RUNNER

A browser-based arcade game built with pure HTML5 Canvas and vanilla JavaScript. No dependencies, no build tools — just open and play.

---

## How to play

You're a spaceship drifting through space. Dodge the incoming meteors — they randomly **split into smaller, faster fragments**. Survive as long as you can.

| Input | Action |
|---|---|
| `↑ ↓ ← →` or `W A S D` | Move |
| `SPACE` | Boost (drains charge, recharges when released) |

The ship rotates smoothly toward your movement direction, diagonals included. Screen edges wrap around.

---

## Features

- Survival timer + local leaderboard saved in `localStorage`
- Your best time is shown every time you open the game
- Increasing difficulty over time
- Particle explosions, animated thrust, boost shockwave effect
- Single `.html` file — no server needed

---

## Run it

Just open `space_survival.html` in any modern browser. That's it.
